# Nodejs_IP_logger
Log Internal and External IP address

install dependencies with:

```npm install```

run the program with:

```npm start```

feel free to modify and share

output of CSV looks like this

| YYYY/MM/DD HH:mm:ss | <EXTERNAL IP> | <INTERNAL IP> |   |   |
|---------------------|---------------|---------------|---|---|
| YYYY/MM/DD HH:mm:ss | <EXTERNAL IP> | <INTERNAL IP> |   |   |
| YYYY/MM/DD HH:mm:ss | <EXTERNAL IP> | <INTERNAL IP> |   |   |
|                     |               |               |   |   |
